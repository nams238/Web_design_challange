Summary: Latitude versus x.



The objective of this project was to analyze how weather changes as you get closer to the equator. To achieve this analysis, weather data were requested from the OpenWeatherMap using the OpenWeatherMap API key to call and assemble a dataset on over 500 cities. These data were processed using Pandas and Matplotlib was used to plot the various aspects of the weather vis-a-vis the latitude. The elements under focus are,temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
